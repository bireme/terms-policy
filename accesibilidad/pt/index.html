ACESSIBILIDADE DOS SÍTIOS E SISTEMAS WEB DA BIREME/OPAS

Todos os sítios e sitemas web da BIREME/OPS seguem as diretrizes W3C para acessibilidade, de acordo com a Web Accessibility Initiative (WAI), cujas recomendações visam tornar a Web acessível a pessoas com deficiência.

METAS ACESSIBILIDADE

Buscamos alcançar o padrão AAA de Diretrizes de Acessibilidade para Conteúdo Web (WCAG 2.0) da W3C e assim fornecer alternativas para que os usuários com necessidades especiais consigam ter uma navegação mais amigável e, por consequência, gerar facilidades para todos que acessam nossos sítios web, não deixando ninguém para trás.

PADRÕES DE TECLAS DE ATALHO

No topo de todos os sítios e sistemas web da BIREME/OPS há uma barra de acessibilidade, na qual os usuários encontram atalhos de navegação, mudança do tamanho de fonte e alto contraste.

Os atalhos são um recurso que permite a navegação nos sítios web com o uso do teclado. Em qualquer página do sítio e sistemas web é possível utilizar as seguintes combinações de teclas:

Teclar Alt + 1, chega-se diretamente ao conteúdo principal da página.

Teclar Alt + 2, chega-se diretamente ao menu principal.

Teclar Alt + 3, chega-se diretamente à caixa de busca.

Teclar Alt + 4, chega-se diretamente ao rodapé da página.

Em caso de Firefox, teclar Alt + Shift + número em vez de Alt + número.

Em caso de Firefox no Mac OS, teclar Ctrl + Alt + número em vez de Alt + Shift + número.

LEIS E POLÍTICAS DE ACESSIBILIDADE DA WEB

Diversos países elaboram leis para atender o público com necessidades especiais, buscando tornar o ambiente web mais democrático e acessível.

Exemplos de leis e políticas estão disponíveis em https://www.w3.org/WAI/policies.

FERRAMENTAS DE AVALIAÇÃO DE ACESSIBILIDADE DA WEB

Há diversas ferramentas disponíveis para avaliar o nível de acessibilidade de páginas web, seguindo as diretrizes WCAG. A W3C lista as ferramentas existentes em https://www.w3.org/WAI/ER/tools/.

A BIREME/OPAS/OMS utiliza como referência a ferramenta https://achecker.ca/checker/index.php.
