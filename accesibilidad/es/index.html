ACCESIBILIDAD DE SITIOS WEB Y SISTEMAS DE BIREME / OPS

Todos los sitios siguen las directrices W3C para accesibilidad, de acuerdo con la Web Accessibility Initiative (WAI), cuyas recomendaciones visan tornar la Web accesible a las personas con discapacidades.

META ACCESIBILIDAD

Buscamos alcanzar el estándar AAA de Directrices de Accesibilidad para Contenidos Web (WCAG 2.0) de W3C y así fornecer alternativas para que los usuarios con necesidades especiales logren tener una navegación más amigable y, por consecuencia, generar facilidades para todos que accedan nuestros sitios web, no dejando a nadie atrás.

PATRONES DE ATAJO DE TECLADO

En la parte superior de todos los sitios hay una barra de accesibilidad, en la cual los usuarios encuentran atajos de teclado para navegación, cambio de tamaño de fuente y alto contraste.

Los atajos son un recurso que permite la navegación en los sitios web a través del uso del teclado. En cualquier página es posible utilizar las siguientes combinaciones de teclas:

Teclear Alt + 1, llegase directamente al contenido principal de la página.

Teclear Alt + 2, llegase directamente al menú principal.

Teclear Alt + 3, llegase directamente a la caja de búsqueda.

Teclear Alt + 4, llegase directamente al pie de página.

En caso de Firefox, teclear Alt + Shift + número en vez de Alt + número.

En caso de Firefox en Mac OS, teclear Ctrl + Alt + número en vez de Alt + número.

LEYES Y POLÍTICAS DE ACCESIBILIDAD DE LA WEB

Diversos países elaboran leyes para atender el público con necesidades especiales, buscando tornar el ambiente web más democrático y accesible.

Ejemplos de leyes y políticas están disponibles en https://www.w3.org/WAI/policies.

HERRAMIENTAS DE EVALUACIÓN DE ACCESIBILIDAD DE LA WEB

Hay muchas herramientas disponibles para evaluar el nivel de accesibilidad de páginas web, siguiendo las directrices WCAG. La W3C dispone un listado de herramientas en https://www.w3.org/WAI/ER/tools/.

BIREME/OPAS/OMS utiliza como referencia la herramienta https://achecker.ca/checker/index.php.
